# Unemployment Analysis Using Python

## Objective
To analyze unemployment trends in India and study the impact of COVID-19 on unemployment rates.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
Unemployment Rate in India Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- State-wise Unemployment Analysis
- COVID-19 Trend Analysis

## Key Insights
- Unemployment rates differed across states.
- COVID-19 significantly affected employment levels.
- Visualization helped identify trends and patterns.

## Result
Successfully analyzed unemployment trends and generated meaningful insights using Python.
